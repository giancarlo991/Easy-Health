Resolução de Bugs Críticos (Banca CI)

1. Windows vs Linux (Case-Sensitivity):

O desenvolvimento local foi feito em Windows (sistema case-insensitive), porém os runners da Action usam Linux (case-sensitive). Imports como require('./models/user') que rodavam localmente causavam erro de compilação Cannot find module no CI. Corrigido mapeando as pastas para minúsculas exatas nos commits #2, #3 e #4.


2. Timeout e Race Condition no Servidor:

A API demorava alguns segundos para conectar ao MongoDB e iniciar. O Newman rodava antes e as requisições quebravam. Corrigido adicionando o utilitário wait-on com timeout de 60s antes de rodar os testes de endpoint.
